# 🛰️ SDR Radar System - نظام رادار كشف الطائرات

[![Python 3.8+](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

نظام رادار متكامل يعمل باستخدام جهاز RTL-SDR للكشف عن الطائرات العسكرية وتحليل الطيف الترددي.

## ✨ المميزات

- 📡 كشف الطائرات عبر نطاق الترددات العسكرية (225-380 MHz)
- 🎨 واجهة رسومية متطورة باستخدام PyQt5
- 📊 عرض الطيف الترددي في الوقت الفعلي
- ✈️ قاعدة بيانات شاملة للطائرات العسكرية
- 🎯 كشف تلقائي للإشارات وتصنيف الطائرات

## 📋 المتطلبات

- Python 3.8 أو أحدث
- جهاز RTL-SDR
- هوائي مناسب لنطاق الترددات العسكرية (225-380 MHz)

## 🚀 طريقة التثبيت والتشغيل

### 1. استنساخ المشروع
\`\`\`bash
git clone https://github.com/your-username/sdr_radar_project.git
cd sdr_radar_project
\`\`\`

### 2. تثبيت المكتبات المطلوبة
\`\`\`bash
pip install -r requirements.txt
\`\`\`

### 3. تشغيل البرنامج
\`\`\`bash
python main.py
\`\`\`

## 🎮 كيفية الاستخدام

1. قم بتوصيل جهاز RTL-SDR بجهاز الكمبيوتر
2. شغّل البرنامج
3. اضغط "بدء الرادار"
4. اضبط التردد على نطاق الطائرات العسكرية (225-380 MHz)
5. راقب الطيف الترددي والقائمة الجانبية للطائرات المكتشفة
6. انقر نقراً مزدوجاً على أي طائرة لعرض معلومات تفصيلية

## 🛠️ التقنيات المستخدمة
